# Two-Factor Authentication (2FA) Security Project

This project is a secure authentication system built with ASP.NET Core that implements Two-Factor Authentication (2FA) to enhance user login security.

The system combines traditional username/password login with an additional verification layer using a time-based one-time password (TOTP).

## Features
- User Registration & Login
- Password Hashing (secure storage)
- Two-Factor Authentication (2FA)
- Time-based One-Time Password (TOTP)
- Login Attempt Logging (success & failed)
- Session Management
- SQL Server Database Integration
- “Remember Device” functionality (planned)
- CAPTCHA verification (planned)


## Technologies Used
- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Google Authenticator (for 2FA)
- Session-based Authentication
